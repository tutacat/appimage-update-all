## appimage-update-all
Simple script to run [AppImage Update Tool](/AppimageUpdate/AppImageUpdate.git) on your programs, or on a specific file.

## Usage
<table><td><code>$ appimage-upall.sh<td><code>$ appimage-upall.sh <file(s)></file></table>

## Config
<td>~./appimage_upall.conf</td>

## Installation
You do not need to install it, you can just download it, but if you want to install it or move the config, move it to the .local/share directory, or another directory of your choosing.
